##	有线网络

###	二层

####		查询端口信息

- **查看某个端口的详细信息**

  ```ini
  display interface GiabitEthernet 1/0/1
  display interface GigabitEthernet 1/0/1 | include input|output|errors
  ```

  可以查看到：接口速率、工作模式、流量统计、错误帧。

- **查看端口流量信息**

  ```ini
  display counters [inbound | outbound | rate] interface GigabitEthernet 1/0/2
  ```

  查看改端口的详细流量计数器（收发字节、包数等）。

- **查看 LLDP邻居（链路层设备发现）**

  ```ini
  display lldp neighbor-information
  ```

  查看当前接口连接到的邻居设备信息，适用于链路排查和拓扑确认。

####	配置相关

- 清空配置

  ```ini
  reset saved-configuration  // 确认删除配置文件（Y）
  reboot                     // 重启设备（不保存当前配置）
  ```



####	管理相关

- 添加用户

  ```ini
  local-user admin class manage
  password simple xxxx
  service-type telnet ssh terminal
  authorization-attribute user-role level-15
  authorization-attribute user-role network-admin
  
  telnet server enable
  ssh server enable
  
  
  # ssh 连接用户的排序，从 0 ~ 5，可供 6 位用户同时登录
  [H3C] line vty 0 5
  [H3C-line-vty0-5] authentication-mode scheme
  # 老设备可能需要以下命令
  [H3C] user-interface vty 0 5
  [H3C] authentication-mode scheme
  
  
  
  # 创建本地用户，并进入本地用户视图
  [H3C] local-user gx class manage

  # 为本地用户授权用户角色
  [H3C-luser-manage-gx] authorization-attribute user-role network-admin
  [H3C-luser-manage-gx] service-type ssh
  [H3C] ssh server enable
  ```

##	无线网络（AC + FIT AP）

####	AP操作

- 重启AP

```ini
<system> reset wlan ap {all | ap-group | model | name | native}
```

在物理 AP 连接的 AC 上执行此命令后，将会重置 AC 与 AP 的 CAPWAP 隧道连接，并重启物理 AP。

####	DHCP 地址池相关 - `display dhcp server ....`

- 检查全局地址池是否有可用的IP地址


```ini
display dhcp server free-ip
```

- 检查是否有过期的IP地址


```ini
display dhcp server expired pool pool-name
```

- 检查是否有冲突的IP地址


```ini
display dhcp server conflict all
```

- 如果存在的冲突IP地址，探测时间没有超过1小时，则执行


```ini
reset dhcp server conflict all
```

如果地址池中的可用IP地址、过期IP地址和冲突的IP都不存在，就说明地址池中IP地址已经用尽.

####	查询 AP 相关信息 - `display wlan ap ....`

|      命令       |                             说明                             |
| :-------------: | :----------------------------------------------------------: |
|       all       |   Display information about about all APs 显示所有AP的信息   |
|     backup      |   AP load balancing and backup status AP负载均衡和备份状态   |
|      name       |          Specify an AP by its name 显示指定AP的信息          |
|   statistics    |                AP statistics 显示AP的统计信息                |
| unauthenticated | Specify all unauthenticated auto APs 显示处于未认证状态自动AP的信息 |

示例：

- 显示某个AP的详细信息


```ini
display wlan ap name f838-8d2a-3100 verbose
```

- 显示某个AP的在线时间


```ini
display wlan ap name f838-8d2a-3100 online-time
```
